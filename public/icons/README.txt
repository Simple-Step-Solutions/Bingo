PWA Icons
=========

This directory should contain production-ready PNG icons before launch:

  icon-192.png  --  192x192 px
  icon-512.png  --  512x512 px

The manifest currently points to /sss-logo.png as a placeholder.
Replace it with properly sized icons and update the manifest icon paths in vite.config.ts.

Recommended tool: https://www.pwabuilder.com/imageGenerator
